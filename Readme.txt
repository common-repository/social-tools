The plugin creates three widgets for displaying various social media sites: Social Icons, Facebook Likebox, Instagram Feed.

== Description ==
Social Tools is a free plugin which creates nice widgets allowing to display links and icons to your social networking  profiles in your WordPress site. Now you can integrate your accounts in the most popular social media (Facebook, Instagram, Twitter, Pinterest, YouTube) into your website and customize the widgets in a few clicks.

The plugin creates the following widgets:

* **Social Tools Facebook Likebox:** here you can add a title and a Facebook page link to show how many likes the page has and how many friends like it.
* **Social Tools Instagram Feed:** here you can add titles, Instagram username or tag, select the number of photos to be shown, choose the size of the photo and the option of opening links in a current window or a new one.
* **Social Tools Social Icons:** here you can add links to your accounts in the most popular social networks - Facebook, Instagram, Twitter, Pinterest, YouTube.

**Features:**
- Fully customizable titles, colors, icon and photo sizes, padding, border radius, etc.
- Input fields for your social profile links
- No need to enter passwords or log into your social accounts
- Shortcodes for creating widgets in any place of your site
- You need only WordPress to use this tool, no additional plugins required
- Compatible with WooCommerce and AliDropship

= Minimum Requirements =

== Installation ==
To install and start using Social Tools, please follow this instruction.
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Appearance` => `Widgets`, search for `Social Tools Facebook Likebox`, `Social Tools Instagram Feed`, `Social Tools Social Icons` and add them to your widget sidebar
4. Choose the options of your choice and insert you social URLs in input boxes
5. Click `Save` button

= What shortcodes can be used to create widgets? =
* An example of the shortcode for Social Icons:
[adswst_social_icons
id="47" // [required field]
facebook_link=""
instagram_link="" 
twitter_link="" 
pinterest_link="" 
youtube_link=""
icon_color="#999999"
icon_color_hover="#FFFFFF"
icon_color_background="transparent"
icon_color_background_hover="#666666" 
icon_font_size="14"
padding="5"
border_radius="2"]

* An example of the shortcode for Instagram Feed:
[adswst_instagram username="@username" size="large" number="5" target="_blank"]
